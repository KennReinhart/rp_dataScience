# Sales Performance Analysis

## Objective
Analyze sales transaction data to understand performance trends, product category contribution, and regional differences.

## Dataset
Transactional retail sales data containing order date, product details, quantity, pricing, and region.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Findings
- Sales increased steadily over the observed period.
- Technology products contributed the highest revenue.
- West and East regions outperformed others.
- Revenue is influenced by a small number of high-value orders.

## Business Recommendations
- Prioritize high-performing product categories.
- Improve sales strategies in underperforming regions.
- Increase average order value through bundling.
- Retain high-value customers.

## Limitations
- Limited time range
- No profitability metrics
- No customer lifecycle analysis

## Next Steps
- Extend analysis to longer time periods
- Incorporate customer segmentation
- Forecast future sales trends
